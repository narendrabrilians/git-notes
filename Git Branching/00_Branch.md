# Branch

## Current Branch

```bash
git branch --show-current
```

## Create Branch

```bash
git branch newbranch
```

## See All Branches

```bash
git branch --list
```

or

```bash
git branch
```

## Switch Branches

```bash
git switch branchname
```

```bash
git checkout branchname
```

## Rename Branch

```bash
git branch -m newbranchname
```

## Delete Branch

```bash
git branch -d branchname
```

```bash
git branch --delete branchname
```
